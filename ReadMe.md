## Installation

git clone git@github.com:Robo-Ryan/ReadMeGenerator.git

<!-- node installation: Navigate to your directory in terminal then input the following-->
npm init -y  
npm i inquirer  
npm i colors  
npm install  
<!-- make sure “type:” “module” are added to the package.json file -->
